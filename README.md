# Overview

This is a simple example implementation of an end-to-end RAG pipeline, including the database embeddings / ingestion and querying.

Since this is an exercise in familiarization with RAG this does not use any pipeline tools like LangChain, only the [Embeddings API](https://developers.openai.com/api/reference/resources/embeddings) from OpenAI and the [Messages API](https://platform.claude.com/docs/en/api/messages/create) from Anthropic.

# Running

1. Set environment variables for `OPENAI_API_KEY`, `ANTHROPIC_API_KEY`, and `DATABASE_URL` in `.env`
2. Update [dummy_corpus.txt](dummy_corpus.txt) if desired (as an example corpus this only has 2 items, I'll leave you to guess which was AI-generated and which was generated by a D&D-loving human person)
3. Run ingestion:

```bash
npx tsx src/ingest.ts dummy_corpus.txt
```

4. Query via CLI:

```bash
npx tsx src/query.ts "Which store has the cheapest product?"
```
